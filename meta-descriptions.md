## Injection Instructions

For each file, paste the block immediately after the existing `<meta charset>` and `<meta viewport>` tags and before `<title>`.

Example placement:
```html
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <!-- PASTE META BLOCK HERE -->
  <title>Page Title</title>
  ...
</head>
```
