DEPLOYMENT GUIDE.

HOMEPORTAL

HOW THE SITE DEPLOYS.

The site is hosted on Vercel and connected to the GitHub repo lonnell-spec/lw-brand-site. Every push to the main branch triggers an automatic deployment. No build step — Vercel serves the static HTML files directly.

TO UPDATE A PAGE.

Option A — GitHub Web Upload:

1. Go to github.com/lonnell-spec/lw-brand-site

2. Click Add file → Upload files

3. Drag the updated file (must have the exact same filename as the one you are replacing)

4. Click Commit changes

5. Vercel auto-deploys in ~30 seconds

CRITICAL RULES.

Filenames matter. The homepage MUST be named index.html — not 1index.html or homepage.html. Vercel looks for index.html at the root.

One file per page. Do not upload versioned copies like 2-index.html. Replace the original file with the same name.

Check after deploy. Visit ldw.build after every push to confirm changes are live.
