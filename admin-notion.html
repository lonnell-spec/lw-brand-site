NOTION
CMS

Notion is the content management system for ldw.build. Two Notion databases feed the site — Content Hub and Photos. Here's how each works.

CONNECTED DATABASES
Content Hub Live Sync
Notion DB ID: f69e7146cb2940ba92d5460001d8611d → Supabase: site_content_hub
Stores content pieces: articles, resources, featured links, and written work. Syncs to the site every 15 minutes via the sync-notion-content edge function.
Required Fields
FieldTypeNotes
TitleTitleRequired — appears as content title on site
TypeSelectarticle / resource / link
StatusSelectSet to "Published" to display on site
URLURLLink for the content piece
DescriptionTextShort description shown on site
Published DateDateUsed for sorting
Photos Live Sync
Notion DB ID: 074724b874a640e1b48cd7667f9addbd → Supabase: site_photos
Stores gallery photos that appear on the homepage. Syncs every 15 minutes. Add a photo row in Notion and it appears on the site automatically.
Required Fields
FieldTypeNotes
TitleTitleCaption or photo title
Photo URLURLDirect image URL (Cloudinary recommended)
CategorySelectspeaking / ministry / personal / event
ActiveCheckboxMust be checked to display on site
Display OrderNumberLower numbers appear first
SYNC SCHEDULE
sync-notion-content (Edge Function v20)
Runs every 15 minutes via Supabase cron. Syncs Content Hub and Photos from Notion. Also runs the YouTube API sync at the same time.
Manual Trigger

GET https://riqdcirsmckubkdfbtoj.supabase.co/functions/v1/sync-notion-content?token=ldw-sync-2026-secure

Note on Instagram and YouTube: These no longer use Notion DBs. Instagram syncs via Zapier (on every new post). YouTube syncs directly from the YouTube API every 15 minutes. The Notion Instagram and YouTube databases were removed in March 2026 to reduce sync complexity.
