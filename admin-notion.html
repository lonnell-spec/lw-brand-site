NOTION SETUP.

The databases that power dynamic content on ldw.build, their sync status, and how to keep them updated.

Architecture

HOW NOTION CONNECTS TO THE SITE

Notion is the source of truth for all dynamic content. The pipeline runs in one direction: Notion → Supabase Edge Function → Supabase database → site reads from Supabase. You update content in Notion, trigger a sync, and the site reflects the change. No code required for content updates.

How to Add Content

UPDATING INSTAGRAM GRID

  1. 01
    Open LDW Site — Instagram database in Notion.
  2. 02
    Add a new row for each post. Required fields: Image URL, Caption, Post Date, Post URL.
  3. 03
    Only the 4 most recent posts display on the site. Order by date descending.
  4. 04
    Trigger a sync from the Admin Portal → Integrations page.

UPDATING CONTENT HUB

  1. 01
    Open LDW Site — Content Hub database in Notion.
  2. 02
    Add entries for sermon clips, podcast episodes, and teaching content. Required: Title, Type (sermon/podcast/teaching), URL, Thumbnail, Date.
  3. 03
    Trigger a sync from Admin Portal → Integrations page.
